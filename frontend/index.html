How we predict

  • We use a Random Forest / Logistic Regression model.
  • Inputs like occurrence time, report time, bike cost, and location coordinates are transformed into numeric features.
  • The model outputs a predicted class and a risk score for STOLEN (not recovered) vs RECOVERED.

Interpreting the result

  • Compare the score to the dataset baseline shown in the result card.
  • A lower score still does not mean "no risk"; it means lower relative non-recovery risk in this dataset.
